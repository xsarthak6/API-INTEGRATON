##This task is based on developing a modern Movie and Series Search Web Application named CineQuest. The main purpose of this project is to help users search for movies, TV series, and episodes quickly by using an online database. The application provides detailed information such as movie title, release year, poster image, genre, director, cast, runtime, IMDb rating, and plot summary. It offers a clean and interactive interface where users can simply enter the name of any movie or series and get instant results. This project demonstrates how front-end web technologies can be integrated with an external API to create a real-time and useful application.

The task was developed using core web technologies such as HTML, CSS, and JavaScript. HTML5 was used to create the structure of the webpage including the search bar, buttons, result containers, cards, and popup modal for movie details. CSS3 was used for designing the user interface with responsive layouts, colors, hover effects, borders, spacing, and animations. The design is modern and user-friendly so that users can easily interact with the application on desktop as well as mobile devices. JavaScript (ES6) was used to handle all the logic of the application such as fetching data from the API, handling button clicks, showing results dynamically, managing errors, and displaying movie details in a popup window.

One of the most important tools used in this task is the OMDb API. This API provides real-time movie and television data from the internet. Using the JavaScript Fetch API, the application sends a request to OMDb whenever the user searches for a movie name. The API then returns data in JSON format, which is processed and displayed on the webpage. This shows how external APIs can be connected to websites to create live applications with updated information.

The editor/platform used for developing this task was Visual Studio Code.
